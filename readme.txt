Registers custom post types and meta boxes for Logus theme.

== Description ==

**The Logus Toolbox plugin is meant to be used only with the ([Logus Theme](https://superadmin.es/logus).**

== Installation ==

1. Go to Plugins > Add New and search for Logus Toolbox. Install and activate it.
2. That's it. No configuration needed.
